# 📝 Text Editor in C++

A simple command-line based Text Editor built using C++. This project demonstrates file handling, string manipulation, and basic data structures.

---

## 🚀 Features

* 📁 Create a new file
* ✍️ Write text into file
* 📂 Open existing file
* 💾 Save file
* 🔍 Search text in file
* 📖 Display file content
* ❌ Delete specific line

---

## 🛠️ Technologies Used

* C++
* File Handling (`fstream`)
* Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Text-Editor-CPP.git
```

2. Navigate to the folder:

```bash
cd Text-Editor-CPP
```

3. Compile the program:

```bash
g++ main.cpp -o editor
```

4. Run the program:

```bash
./editor
```
